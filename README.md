# BookingCore Integration

Этот проект демонстрирует, возможную доработку в систему **BookingCore**.

## Возможность добавления (Теория)

1. **Добавить в ServiceProvider**
2. **Зарегистририовать миграции**
3. **Подключить API маршруты** 
4. **Зарегистрировать политику доступа**
5. **Добавить авторизацию к эндпоинтам**
6. **Валидацию прав доступа к данным**
7. **Логирование операции**
8. **В зависимости от дополнительного функционала добавить соответствующие функции**
